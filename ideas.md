### Idea A — "Terminal Clarity" (Dark Technical Aesthetic)
**Design Movement:** Neo-brutalist data terminal meets modern developer portfolio
**Core Principles:**
- Dark navy/charcoal background with high-contrast teal/cyan text — feels like a real data environment
- Monospace fonts for code sections, sharp sans-serif for prose — clear hierarchy between "data" and "narrative"
- Sections feel like dashboard panels with subtle borders and glow effects
- Metrics and numbers are the heroes — large, bold, glowing

**Color Philosophy:** Deep navy (#0F2D4A) base, electric teal (#06B6D4) for accents, off-white (#E8F4F8) for body text. Evokes the feeling of working inside a real analytics tool.

**Layout Paradigm:** Left-anchored vertical nav rail + scrolling main content. Each section is a "panel" with a header label. Not a centered hero — feels like you're inside the tool.

**Signature Elements:**
- Animated number counters for key metrics (100+ hrs, 29 widgets, etc.)
- SQL code blocks with real syntax highlighting in a terminal-style container
- Dashboard screenshots in a "screen frame" with subtle glow

**Interaction Philosophy:** Hover states that feel like activating a dashboard widget. Smooth scroll between sections. Code blocks with copy button.

**Animation:** Entrance animations on scroll — panels slide in from left. Number counters animate up on first view. Subtle teal pulse on metric badges.

**Typography:** `JetBrains Mono` for code/labels, `DM Sans` for body — clean, technical, readable.

### Idea B — "Editorial Data" (Magazine + Data Visualization Hybrid)
**Design Movement:** Data journalism aesthetic — think The Pudding or FiveThirtyEight
**Core Principles:**
- Light background with strong typographic hierarchy — feels authoritative and editorial
- Data visualizations are embedded inline with narrative, not siloed
- Generous whitespace with sharp typographic contrast
- Color used sparingly but with high intentionality

**Color Philosophy:** Warm white (#FAFAF8) base, deep navy (#0F2D4A) for headings, teal (#0E7490) for data accents, amber (#F59E0B) for highlights. Feels credible and analytical.

**Layout Paradigm:** Asymmetric editorial grid — wide left column for narrative, narrow right column for callout stats and annotations. Breaks the typical centered portfolio mold.

**Signature Elements:**
- Pull-quote style metric callouts (large number + small descriptor)
- Section dividers that look like chart gridlines
- "Annotation" style labels on dashboard screenshots

**Interaction Philosophy:** Minimal animation — content is the focus. Hover reveals additional context on metrics. Smooth section transitions.

**Animation:** Subtle fade-in on scroll. Metric numbers count up once. No flashy transitions — data speaks.

**Typography:** `Playfair Display` for section headings, `Source Sans Pro` for body — editorial gravitas with readability.

### Idea C — "Precision Dashboard" (Dark Mode Analytics UI)
**Design Movement:** Enterprise analytics platform aesthetic — Looker/Hex meets personal portfolio
**Core Principles:**
- Dark charcoal (#1A1F2E) background with card-based layout — feels like an actual dashboard
- Teal/cyan primary color with subtle grid texture background
- Cards have soft shadows and subtle borders — depth without noise
- Typography is clean and functional — not decorative

**Color Philosophy:** Dark slate (#1A1F2E) background, card surfaces (#242938), teal (#0EA5E9) primary, muted blue-gray for secondary text. Feels like a premium BI tool.

**Layout Paradigm:** Card grid layout — each project/section is a "widget" in a dashboard. Top nav with section anchors. Sidebar stat panel on wider screens.

**Signature Elements:**
- KPI stat cards at the top (15 yrs experience, 29 widgets built, 100+ hrs saved)
- Dashboard screenshots inside realistic browser/app chrome frames
- Skill tags styled like filter chips in a BI tool

**Interaction Philosophy:** Cards lift on hover with shadow increase. Smooth scroll. Active nav indicator follows scroll position.

**Animation:** Cards animate in on scroll with a subtle upward drift. KPI numbers count up. Nav indicator slides smoothly.

**Typography:** `Space Grotesk` for headings, `Inter` for body — clean, modern, technical.

## Selected Approach: **Idea A — "Terminal Clarity"**

Dark navy base with electric teal accents. Left-rail nav + panel-based scrolling content. JetBrains Mono for code, DM Sans for prose. Animated metric counters, terminal-style SQL blocks, dashboard screenshots in glowing frames. This approach is the most distinctive and directly signals "I live in data tools" to a hiring manager — while still being polished and readable.
